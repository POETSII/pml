### Requirements

`pml` requires Python 2 and `pip`.

### Installation

Using `pip`:

```bash
git clone https://github.com/POETSII/pml.git
pip install -r requirements.txt
```

or with `pip` and `virtualenv`:

```bash
git clone https://github.com/POETSII/pml.git
virtualenv env
source env/bin/activate
pip install -r requirements.txt
```
